CRAFTER☰

A MINECRAFT PACKAGE MANAGER

WHAT IT IS

Crafter is a package manager. What is a package manager, you may ask? A package manager is a program that, well, manages packages—that is to say, it automates installing things. So, on Debian-based systems, instead of downloading and running the installer for GParted, you could run sudo apt-get install gparted in a terminal and your package manager will automagically download and install everything needed for GParted.

How does this apply to Minecraft? Well, imagine that instead of digging through forums and wading through Adf.ly links, all you had to do to install a mod was crafter install. Would that be awesome? Of course! That is exactly what Crafter does.

HOW IT WORKS

Crafter hooks into a massive mod repository called HeisenBugDev. Modders host their mods on HeisenBugDev so that they can be distributed more easily. Plus, the modders get a cut of the ad revenue.

Then, various launchers like Crafter hook into HeisenBugDev to get the mods needed. Crafter is different from all the others because it has a CLI and a GUI, which means that anyone can use the GUI to install mods, but superusers can use the more powerful CLI.

MODPACKS

By now, you're probably wondering: What about modpacks? We've come up with a fantstic system for that: Craftfiles. All you have to do to install a modpack is crafter install <modpack name>. Crafter will download the Craftfile, parse it, and download and configure all the required mods. Craftfile markup is the simplest possible: it's just Ruby!

Remember that GUI we mentioned earlier? You can generate Craftfiles with it, too. Here's the markup:

                minecraft '1.6.4'
forge     '1234'
          # Other options are "latest" or "recommended" (default)

source 'heisenbugdev' # some sugar for http(s)://heisenbugdev.com
source 'github://ZephireNZ@Quickmods/index.json' # A quickmod index (syntax is for MultiMC compat)
source 'https://github.com/luacs1998/QuickModFiles' # This will automatically detect the index.json

mod 'QuantumCraft' # gets latest QuantumCraft from heisenbugdev
mod 'SpongesMod', '@1.6.4'
                  # Any version for 1.6.4 (1.6 will match to 1.6.x)
                  # Also valid would be '>=2.0.2' and other normal matchers
                  # Defaults to current MC version (Ignore the last digit?)

mod 'ProjectRed', github: 'MrTJP/ProjectRed', ref: 'v1.0'
                                              # Ref can either be a git tag,
                                              # commit, or branch

# Aliased to rp
resourcepack '', url: 'http://dropbox/.../bleh.zip'
